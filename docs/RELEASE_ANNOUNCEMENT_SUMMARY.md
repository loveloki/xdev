## 🎯 核心价值

### 用户导向
- 描述用户能做什么，而不是开发团队做了什么
- 关注用户体验的改进，而非技术实现细节
- 使用"你"来直接与用户对话

### 突出价值
- 只包含对用户体验影响最大的功能
- 避免"各种改进和错误修复"等模糊描述
- 优先展示用户工作流程的改进

### 积极表达
- 说"更快"而不是"不那么慢"
- 关注性能提升而非bug修复
- 用积极的语言描述变化
